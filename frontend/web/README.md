# Guard Dispatch — Admin Portal

Next.js admin dashboard for the Security Guard Dispatch System.

## Development

```bash
cd frontend/web
npm install
npm run dev
```

Open [http://localhost:3000/p-guard-app](http://localhost:3000/p-guard-app)

## Docker

```bash
docker compose up web-admin
```

## Tech Stack

- Next.js 16 (App Router, Standalone output)
- React 19
- Tailwind CSS 4
- TypeScript 5
- Lucide React Icons
